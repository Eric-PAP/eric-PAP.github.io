Turn based RPG

Being a big fan of the game Dofus by Ankama, I created this prototype where you can fight monsters on multiple level with your 3 playables characters!

Tools

GameMaker: Studio | Tiled

Work

Level creation workflow

  • Create the level with Tiled
  • Save the level as a .json file
  • Custom script to import it

Characters

  • 3 characters
  • Action and movement points
  • 15 spells per character unlocked by leveling up
  • Spritesheet animation (source)

A multitude of spells

  • Single target or AoE
  • Custom range
  • With or without line of sight
  • Attack only in line or not
  • Attacks that moves the player or target
  • Buffs, debuffs, heals, traps and more!

Basic AI

  • Use A* algorithm for pathfinding
  • Attack when in range

User interface

Rough UI where you can:
  • See the play order
  • See your remaining action points
  • Select the spells to cast
  • End your turn
